# Hooks Food App – Full Stack Food Ordering Platform

## 🌐 Live Demo
https://food-app.iraynetworking.org  

## 🛠 Tech Stack
- Frontend: React.js, Vite, Tailwind CSS
- Backend: Python FastAPI
- Database & Auth: Supabase (PostgreSQL)
- Payment Integration: Paystack
- Deployment: Production Cloud Hosting

## 🚀 Features
- User authentication with Supabase
- Restaurant menu browsing
- Dynamic cart system
- Order creation & tracking
- Paystack payment gateway integration
- Admin order management logic
- Responsive mobile-first UI
- Structured restaurant and menu system

## 🏗 Architecture
Frontend (React + Tailwind CSS)  
→ Backend API (FastAPI)  
→ Supabase (Database + Authentication)  
→ Paystack (Secure Payments)  
→ Cloud Deployment
