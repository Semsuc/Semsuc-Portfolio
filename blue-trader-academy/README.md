## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Media & Storage: Cloudinary
- Deployment: Render / Hostinger

## 🚀 Features
- User authentication & account management
- Course browsing and enrollment
- Upload & manage learning content
- Admin dashboard for managing users and courses
- File uploads managed via Cloudinary
- Responsive UI for web and mobile

## 🏗 Architecture
Frontend (React + Tailwind CSS) → Backend (Node.js/Express) → MongoDB → Deployment (Render/Hostinger)
