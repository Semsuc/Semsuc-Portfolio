
# Timah’s Kitchen – Full Stack Food Ordering System

## 🌐 Live Demo
[https://timahskitchen.co.uk](https://timahskitchen.co.uk)  

## 🛠 Tech Stack
- Frontend: React.js, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Media & Storage: Cloudinary
- Deployment: Render, Hostinger
- Payment: Stripe-like gateway (secure online payments)

## 🚀 Features
- User authentication & account management
- Browse menu items with categories
- Cart system for orders
- Online payment integration
- Admin dashboard to manage orders & products
- Order tracking
- Image upload & optimization via Cloudinary
- Responsive UI for mobile & desktop

## 🏗 Architecture
Frontend (React + Tailwind CSS) → Backend (Node.js/Express) → MongoDB → Payment Gateway → Deployment (Render/Hostinger)
